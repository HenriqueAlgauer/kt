Objetivo: Desenvolver um módulo para criação de personagem seguindo as regras do D&D (arquivo já enviado no Black Board) e em linguagem Kotlin.

Requisitos:

- O projeto deve estar em linguagem kotlin, utilizando build em Maven ou Gradle, no IntelliJ, e disponibilizado no seu repositório github (autorizem meu acesso incluindo o e-mail aryelmarlus@hotmail.com).
- O projeto deve possuir ao menos uma abordagem utilizando o padrão de projeto Strategy.

- O personagem deve possuir os atributos descritos no livro (Força, Destreza, Constituição, Inteligência, Sabedoria, Carisma) e também os pontos de vida (utilizar um valor padrão de 10 + modificador de constituição).

- O personagem deve possuir os bônus raciais de cada uma das raças (implementar todas).
- Não é obrigatório se preocupar com as classes neste momento.
- Junits não são obrigatórios, mas recomendados e serão um ponto extra.
